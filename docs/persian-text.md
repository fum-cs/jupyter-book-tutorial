## نوشتن متن فارسی

برای نوشتن متن فارسی کافی است متن خود را به صورت معمولی وارد کنید. برای نمایش صحیح راست‌به‌چپ، از تگ `<div dir="rtl" markdown="1">` استفاده کنید:

```markdown
<div dir="rtl" markdown="1">

متن فارسی شما اینجاست.

- مورد اول
- مورد دوم

</div>
```

خروجی:

متن فارسی شما اینجاست.

- مورد اول
- مورد دوم
